# 🌿 Koru

**Koru** is an AI-powered adaptive learning platform built for hackathon prototyping.  
Enter any topic—many as you want—and let the AI agent craft a complete, structured learning experience. Each topic gets its own Elo rating that evolves with your performance, all managed from a beautiful dashboard of topic cards.

> *Koru* (Māori for “loop” or “spiral”) symbolizes continuous, expanding growth—just like your knowledge.

---

## ✨  Core Idea

1. **User inputs any topic** – “Quantum Computing”, “Renaissance Art”, “TypeScript Generics”… add as many as you like.
2. **AI agent generates** learning objectives, notes, flashcards, and a quiz for that topic – output as structured `.json` / `.yaml` files.
3. **A new topic card** is created on your dashboard and assigned a default Elo rating (1000).
4. You study the materials and answer quiz questions.
5. The topic’s Elo updates based on your performance, and the AI agent **adapts future materials** to match.
6. Your dashboard gives you an at-a-glance view of every topic, each with its current Elo and progress.

- **Data Formats**: JSON, YAML (generated by the AI)

🤖 AI Agent Details
The AI agent generates a complete topic package.
It outputs four files per topic:

learning_objectives.json – list of objectives with descriptions and difficulty levels

notes.yaml – structured study notes

flashcards.json – Q&A pairs for active recall

quiz.yaml – multiple-choice/true-false questions with correct answers and explanations

These files are parsed by the frontend to render the learning interface for each topic.

Adaptive Elo System (Prototype)
Every new topic starts with an Elo of 1000.

As you answer quiz questions, the topic’s Elo changes based on correctness and confidence.

When you revisit a topic or request more materials, the AI agent receives the topic’s current Elo as context.

This allows the agent to adjust difficulty, depth, and complexity dynamically.

Example prompt snippet:

text
The user’s current Elo for “Quantum Computing” is 1250.
Generate a new set of advanced flashcards and a quiz that match this level.
🎨 UI / Design
Built with shadcn/ui and Radix primitives for a clean, accessible, and responsive interface.
Tailwind CSS handles all styling.

Dashboard – a grid of topic cards, each showing the topic name, a sparkline or badge with its current Elo, and quick access to materials.

Topic View – tabbed interface for objectives, notes, flashcards, and quiz.

Creation Flow – simple input field to add a new topic, with instant generation feedback.

🔮 Hackathon Notes
This is a prototype – meant to demonstrate the core loop of AI‑generated adaptive learning across multiple topics.

Data is ephemeral (in‑memory or local storage) for quick iteration.

File generation (JSON/YAML) is simulated or backed by server-side API calls.

The adaptive Elo logic is a simplified model – real implementations would involve spaced repetition and more nuanced performance tracking.
